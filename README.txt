README FOR Custom Form Elements
****************************************************

1. What is/are Custom Form Elements?
°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°
Custom Form Elements unites all efforts in the web to enhance web-based XHTML forms
in terms of style, usability and accessibility by using Javascript and/or CSS.
Basically, it's a UI library.

2. Requirements to use CFE
°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°
JavaScript-compatible Browser see => http://customformelements.net for a list of supported browsers

Custom Form Elements are based on various Javascript Framworks. Currently these are supported:
    * Mootools 1.1 [deprecated]
    * Mootools 1.2
